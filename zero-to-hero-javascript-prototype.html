Introdução

Em JavaScript a herança que tanto utilizamos em linguagens como Java ou C# é um pouquinho diferente. Basicamente todo objeto possui uma cadeia de protótipos e suas propriedades são herdadas pelo objeto.

Neste artigo iremos aprender a manipular protótipos junto de suas principais características.

Manipulação através de proto

A primeira e mais simples forma de manipulação de protótipos se dá através da propriedade proto.

Exemplo: 

    var pessoa = {
      sexo: 'Masculino'
    };

    var renan = {
      nome: 'Renan Johannsen de Paula',
      idade: 24,
      __proto__: pessoa
    };

    pessoa.sexo;
    // $ Masculino

Vale ressaltar que a propriedade proto não é suportada por todos os interpretadores, portanto fique de olho.

Manipulação através de Object.prototypeOf

Uma forma mais elegante de manipulação do protótipo de objetos se dá através dos métodos getPrototypeOf e setPrototypeOf contidos em Object.

Exemplo: 

    var pessoa = {
        sexo: 'Masculino'
    };

    var renan = {
        nome: 'Renan Johannsen de Paula',
        idade: 24
    };

    Object.setPrototypeOf(renan, pessoa);

    pessoa.sexo;
    // $ Masculino

Manipulação através de Object.Create

Outra forma de atribuição de protótipos é através do método create contido em object. O mesmo recebe como parâmetro o protótipo a ser referenciando pelo objeto a ser criado.

Exemplo: 

    var pessoa = {
        sexo: 'Masculino'
    };

    var renan = Object.create(pessoa);
    renan.nome = 'Renan Johannsen de Paula';
    renan.idade = 24;

    pessoa.sexo;
    // $ Masculino

Prototype em funções

Funções em JavaScript possuem uma propriedade prototype que referencia um objeto vazio, ou seja, quando estamos utilizando funções construtoras a forma de se trabalhar com herança se dá através do acesso a essa propriedade.

Exemplo: 

    function Pessoa(nome, idade) {
        this.nome = nome;
        this.idade = idade;
    }

    Pessoa.prototype.sexo = 'Masculino';

    var renan = new Pessoa('Renan Johannsen de Paula', 24);

    renan.sexo;
    // $ Masculino

Shadowing

Uma característica da linguagem é que a busca por propriedades dentro dos objetos se dá de forma recursiva, ou seja, as propriedades do objeto serão priorizadas perante ao protótipo.

Ao atribuirmos um valor a propridade herdado do objeto o valor do protótipo não será sobrescrito, mas sim o objeto terá uma propriedade sombra do protótipo.

Exemplo: 

    var pessoa = {
      sexo: 'Masculino'
    };

    var renan = Object.create(pessoa);
    renan.nome = 'Renan Johannsen de Paula';
    renan.idade = 24;
    renan.sexo = 'Teste';

    console.log(renan);
    // $ { nome: 'Renan Johannsen de Paula', idade: 24, sexo: 'Teste' }

    console.log(renan.sexo);
    // $ Teste

    console.log(Object.getPrototypeOf(renan).sexo);
    // $ Masculuno

Vale ressaltar que a criação de uma cadeia muito extensa de protótipos pode prejudicar o tempo de acesso as propriedades, principalmente se estiverem no topo da cadeia.
